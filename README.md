# PourLog MVP

와인 사진 인증 피드, 댓글, 선입금 기반 와인 모임 모집을 검증하기 위한 정적 웹앱 프로토타입입니다.

## 실행

브라우저에서 `index.html`을 열면 바로 실행됩니다. 현재 버전은 백엔드 없이 `localStorage`에 데이터를 저장합니다.

## MVP 범위

- 카카오 로그인 플로우를 연결하기 전의 로그인 상태 시뮬레이션
- 와인 사진, 와인명, 장소, 자랑글 작성
- 피드 댓글 작성
- 와인 모임 생성
- 선입금 금액, 계좌, 예금주 정보 표시
- 참가 신청 시 모집 인원 증가
